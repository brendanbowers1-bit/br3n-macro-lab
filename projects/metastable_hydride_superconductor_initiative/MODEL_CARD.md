# Model Card: Metastable Hydride Candidate Screening

## Model / Workflow Name

Metastable Hydride Candidate Screening Workflow

## Purpose

Evaluate hypothetical hydrogen-rich derivative hydrides for structural plausibility, dynamic stability, electronic interest, and synthesis feasibility.

## Initial Candidate

**Parent:** Mg₂IrH₅

**Hypothetical derivative:** Mg₂IrH₆

## Inputs

- parent crystal structure
- proposed hydrogen insertion sites
- DFT settings
- pseudopotentials
- competing phase references
- phonon calculations
- electronic structure outputs
- literature constraints

## Outputs

- relaxed structures
- formation energy estimates
- phonon stability result
- density of states
- electron-phonon coupling estimate if applicable
- candidate ranking
- uncertainty notes

## Intended Use

- research planning
- computational screening
- lab outreach preparation
- hypothesis prioritization

## Not Intended For

- claiming discovery
- claiming superconductivity
- DIY synthesis
- safety-free experimentation
- publication without validation

## Known Limitations

- DFT functional sensitivity
- pseudopotential dependence
- missing competing phases
- metastability uncertainty
- high-pressure synthesis uncertainty
- hydrogen position uncertainty
- electron-phonon calculation cost
- experimental feasibility unknown

## Validation Plan

- reproduce parent phase
- test multiple candidate structures
- compare competing phases
- run phonons
- run electron-phonon coupling only after stability screening
- obtain expert review
- require independent replication

## Disclaimer

This workflow supports speculative research planning only.
